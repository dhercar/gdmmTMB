# Centered title with decorated side

Centered title with decorated side

## Usage

``` r
print_title2(text, symb = "-", width = getOption("width"))
```

## Arguments

- text:

  Title to display

- symb:

  Symbol used left and right of the title

- width:

  Size of the console. Defaults to current width.

## Value

A `character` string (invisibly returned)
