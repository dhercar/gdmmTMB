# Print method for bbgdmm objects

Prints basic information from a fitted bbgdmm object

## Usage

``` r
# S3 method for class 'bbgdmm'
print(x, ...)
```

## Arguments

- x:

  A fitted model object of class `bbgdmm`

- ...:

  Additional parameters (not used)
