# Print method for gdmm objects

Prints basic information from a fitted gdmm object

## Usage

``` r
# S3 method for class 'gdmm'
print(x, ...)
```

## Arguments

- x:

  A fitted model object of class `gdmm`

- ...:

  Additional parameters (not used)
